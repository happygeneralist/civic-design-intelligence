# Shared Obsidian Bases

Use this folder for shared repository views that help contributors inspect the knowledge base consistently.

Shared bases should be named for their purpose, for example `User needs.base` or `Review queue.base`.

Personal views should stay local. Use root-level `Untitled*.base`, `*.local.base` or a local-only folder such as `000_Local/` for experiments that should not be committed.
