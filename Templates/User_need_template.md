# User need title

Follow `docs/User_needs_writing_rules.md`, `docs/Needs_hierarchy.md` and `docs/Lifecycle_states.md`.

## Working formulation

Write the need clearly and solution-agnostically.

Avoid cognitive verbs such as `know` or `understand` unless comprehension is genuinely the thing the person must achieve.

```text
I need to [action] [object/context] so that [outcome/value].
```

Do not force this structure where it would distort an experience, civic, emotional or social need.
